## Instalar y ejecutar

```bash
# Install
$ npm install

# Run
npm run dev
```

## Uso

El proyecto expone tres endpoint basicos:

- `/api/helloworld` Para testear que nuestro proyecto esta funcionando
- `/api/users`
- `/api/products`

### Ejemplo basico **Crear USUARIO** `/api/users`:

Request Body:
```json
{
  "name": "alexis",
  "email": "alexis@gmail.com",
  "password": "my-secret-password",
  "role": "user"
}
```
